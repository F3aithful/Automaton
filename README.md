Проект Automaton содержит упражнения и лабораторные работы по автоматизации тестирования, разделённые по урокам. Каждая папка с уроком включает тесты, примеры использования Pytest и подготовку к формированию отчетов в Allure.

🧩 Структура репозитория

lesson1/lesson_01 — базовые примеры тестирования и структуры тестов.

lesson2/lesson_02 — продолжение обучения: простые API-тесты.

lesson3/lesson_03 — работа с параметризованными тестами и фикстурами.

lesson4/lesson_04 — тестирование более сложных сценариев.

lesson5/lesson_05 — расширенные методы автоматизации.

lesson6/lesson_06 — интеграционные тесты и работа с данными.

lesson6K/06K_lesson — дополнительные задания по уроку 6.

lesson7/07_lesson — тестирование веб-интерфейсов.

lesson8/08_lesson — работа с API и JSON-ответами.

lesson9/09_lesson — комбинированные тесты и отчеты.

leccon10/lesson_10 — финальные задания и примеры отчетности.

🚀 Возможности

Написание ручных и автоматизированных тестов с Pytest.

Использование фикстур и параметризации для повторного использования кода.

Формирование Allure отчетов для визуализации результатов тестирования.

Тестирование API веб-приложений, проверка JSON-ответов и различных сценариев.

⚙️ Установка и запуск

Клонируйте репозиторий:

git clone https://github.com/F3aithful/Automaton.git

Установите зависимости:

pip install -r requirements.txt

Запуск тестов:

pytest -v

Генерация Allure отчета:

allure serve ./allure-results
🗂️ Примеры

Каждая папка с уроком содержит тестовые сценарии, которые можно запускать отдельно или по всему проекту для практики и экспериментов.
